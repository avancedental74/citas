Conecta Google Analytics 4

Para ver los datos de clics en Google Reviews necesitas crear una Service Account en Google Cloud y darle acceso a tu propiedad de GA4. Es un proceso de 5 minutos.

  1. Ve a console.cloud.google.com → Crea un proyecto
  2. Habilita la API: Google Analytics Data API
  3. Crea una Service Account y descarga el JSON de credenciales
  4. En analytics.google.com → Admin → Gestión de acceso → añade el email de la Service Account como Viewer
  5. Pon la ruta al JSON y el Property ID abajo y pulsa Guardar
